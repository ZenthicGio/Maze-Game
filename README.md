This project started as a maze generator, which quickly evolved into a full-fledged video game (it's the first video game I've programmed, and it's a prototype). First, I created the maze generator, then I added the player, then the enemies. I added a scoring system, a weapon system, and enemy AI for pursuit. I later changed everything to make this game a little more immersive and interesting: no external UI, I added menus, and made the gameplay dynamic. Currently, it's Windows-only (for saves and runtime), but I'll soon make this indie game available for both Mac and Linux.

For anyone who has played or will play this game: I know it's not the best video game, but it's my very first video game programmed entirely in JS/HTML/CSS and converted with Neutralino to run as an EXE.
